Local Setup for Live AI

For the complete experience with live AI chat:

  1. Clone the repository from GitHub
  2. Navigate to the demo directory
  3. Run a local server: python3 -m http.server 8088
  4. Visit: http://localhost:8088/working-demo.html

This local setup ensures your API keys remain secure and avoids cross-origin resource sharing (CORS) issues.
